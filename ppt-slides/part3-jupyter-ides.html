Module 03

Jupyter & Modern IDEs

Data science is experimental. Traditional scripts aren't enough. Today we master Interactive Computing and professional development environments.

The Concept

Why Notebooks?

Notebooks allow you to combine Live Code, Visualizations, and Markdown in a single document (.ipynb).

Environment Choice

The Dual Path

VS Code: Best for full projects, debugging, and git integration.
Jupyter Lab: Best for pure data exploration and lightweight research.
Setup

Configuring VS Code

Install the Jupyter Extension from the Marketplace. This allows VS Code to render .ipynb files natively.

First Steps

Creating .ipynb

In your terminal or IDE, create a new file named experiment.ipynb. VS Code will automatically open the notebook interface.

Configuration

Selecting the Kernel

Look at the top-right corner. Click "Select Kernel" and choose your ml_internship environment created in Part 1.

Basics

Code vs Markdown

  • Code Cells: Executable Python logic.
  • Markdown Cells: Documentation, headers, and math equations (LaTeX).
Workflow

The Execution Order

Notebooks are stateful. If you run cell 2 before cell 1, variables defined in cell 1 will be missing. Always track the In [n] number.

Shortcuts

Running Cells

Shift + Enter : Run cell and move to next.
Ctrl + Enter : Run cell and stay on current.
Shortcuts

Command Mode

Press Esc to enter Command Mode, then use:

  • A : Insert cell Above
  • B : Insert cell Below
  • D, D : Delete cell
Jupyter Magics

Timing Code

Use %%time at the top of a cell to see how long your ML model takes to train.

%%time
# Some heavy ML training here
model.fit(X, y)
Visualization

Inline Plots

Notebooks render charts directly below the code. No need to save images to disk to view them.

Productivity

Terminal Access

You can run terminal commands inside a notebook by prefixing them with !.

!pip install seaborn
!ls -lh
Troubleshooting

The "Panic" Button

If your code loops infinitely or memory fills up, use Restart Kernel. This clears all variables but keeps your code.

Cloud Options

Google Colab

Zero-setup cloud notebooks. Includes free access to GPUs (critical for Part 4/Deep Learning).

IDE Powerups

The ML Stack

  • Pylance: Type checking and auto-completion.
  • Python Indent: Automatic formatting.
  • Data Wrangler: Visual data cleaning (Built into VS Code).
Debugging

The Variable View

In VS Code, click the "Variables" button in the top toolbar to see all active DataFrames and arrays in memory.

Checklist

Module Validation

  • [ ] Successfully ran a "Hello World" cell

Pro Tip

Notebook Cleanliness

Rule: Before sharing a notebook, run "Restart & Run All" to ensure your results are reproducible from start to finish.
