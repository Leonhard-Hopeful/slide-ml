Module 04

The ML Library Stack

Python's strength in AI comes from its ecosystem. Today we explore the specialized tools for math, data management, modeling, and visualization.

Overview

The Ecosystem

NumPy: Numerical foundation.
Pandas: Data manipulation.
Scikit-Learn: Machine Learning.
Matplotlib: Data visualization.
NumPy

The Math Engine

NumPy (Numerical Python) is the foundation. It provides the ndarray object which is significantly faster and more memory-efficient than standard Python lists.

NumPy

Arrays vs Lists

PYTHON
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
# Vectorized operation (Fast)
print(arr * 2) # [2, 4, 6, 8, 10]
NumPy

Matrices & Tensors

In ML, images and data are matrices. NumPy makes handling them simple.

matrix = np.array([[1, 2], [3, 4]])
print(matrix.shape) # (2, 2)
Pandas

The Data Architect

Pandas introduces the DataFrame—essentially an Excel spreadsheet for Python. It is the go-to tool for cleaning and exploring datasets.

Pandas

Loading Data

PYTHON
import pandas as pd

df = pd.read_csv('data.csv')
print(df.head()) # Preview first 5 rows
Pandas

Instant Insights

Ask questions about your data instantly:

df.describe() # Statistical summary
df['target'].value_counts() # Class distribution
Scikit-Learn

The ML Toolkit

Scikit-Learn (sklearn) provides a consistent interface for hundreds of algorithms. If it's a standard ML task, sklearn has a tool for it.

Scikit-Learn

Fit, Predict, Score

The standard workflow for almost every model:

1. model.fit(X, y): Training
2. model.predict(X_new): Inference
3. model.score(X, y): Evaluation
Scikit-Learn

Linear Regression

from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Scikit-Learn

Feature Scaling

ML models perform better when data is on the same scale (e.g., 0 to 1).

from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
Matplotlib

The Canvas

The low-level plotting library. It gives you complete control over every pixel in your graph, chart, or figure.

Matplotlib

Basic Line Plot

PYTHON
import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [10, 20, 30])
plt.title("Growth over Time")
plt.show()
Seaborn

Statistical Style

Built on top of Matplotlib, Seaborn makes complex statistical charts (like heatmaps or violin plots) beautiful with very little code.

Seaborn

Correlation Matrix

import seaborn as sns
sns.heatmap(df.corr(), annot=True)
Decision Guide

Which tool when?

  • Numbers/Math? Use NumPy.
  • Spreadsheets/CSVs? Use Pandas.
  • Training Models? Use Scikit-Learn.
  • Graphs/Charts? Use Matplotlib/Seaborn.
Verification

  • [ ] Understands the Fit/Predict pattern
  • [ ] Can create a simple plot
