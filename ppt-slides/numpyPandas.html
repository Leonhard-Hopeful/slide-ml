02 | NumPy

Numerical Computing Foundation

NumPy is the "engine" under the hood of almost every Python data library.

  • Optimized C-compiled backend
  • Memory efficient storage
  • Vectorized operations (No loops!)
02 | NumPy

Installation & Import

TERMINAL & PYTHON
# Install via pip
pip install numpy

# Standard industry import convention
import numpy as np
02 | NumPy

Creating Arrays

NUMPY ARRAYS
# Create a 1D array from a list
arr = np.array([1, 2, 3, 4])

# Create a 2D matrix
matrix = np.array([[1, 2], [3, 4]])

print(arr)
print(matrix)
02 | NumPy

Why Not Python Lists?

NumPy arrays are Homogeneous (one type), allowing the CPU to process them as a single continuous block of memory.

02 | NumPy

Vectorized Computation

NO LOOPS REQUIRED
arr = np.array([1, 2, 3, 4])

print(arr * 2)  # Multiplies every element by 2
print(arr + 5)  # Adds 5 to every element
print(arr ** 2) # Squares every element
02 | NumPy

Matrix Operations

matrix = np.array([[1, 2], [3, 4]])

print(matrix.shape)  # Returns (2, 2)
print(matrix.T)      # Transposes the matrix
02 | NumPy

Indexing & Slicing

arr = np.array([10, 20, 30, 40, 50])

# Access by index
print(arr[0])   # 10

# Slice (start:stop)
print(arr[1:4]) # [20, 30, 40]
02 | NumPy

Statistical Power

data = np.array([10, 20, 30, 40, 50])

print(np.mean(data))   # Average
print(np.std(data))    # Spread
print(np.median(data)) # Middle value
02 | NumPy

Understanding Broadcasting

Broadcasting describes how NumPy treats arrays with different shapes during arithmetic operations.
02 | NumPy

Applicability Check

When should you reach for NumPy?

  • Neural network weights manipulation
  • Image processing (pixels as arrays)
  • Signal processing (waves as arrays)
  • Complex Linear Algebra
03 | Pandas

Structured Data Mastery

If NumPy is the engine, Pandas is the car's interior. It makes data "human-readable" in tables called DataFrames.

03 | Pandas

Installation & Import

pip install pandas

import pandas as pd
03 | Pandas

Creating DataFrames

data = {
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [25, 30, 35],
    "Salary": [50000, 60000, 70000]
}

df = pd.DataFrame(data)
# This creates a structured table with indexes
03 | Pandas

Reading External Files

# The most common industry tasks
df = pd.read_csv("data.csv")
df_excel = pd.read_excel("data.xlsx")
03 | Pandas

Initial Inspection

df.head()     # Peek at first 5 rows
df.info()     # Check data types and nulls
df.describe() # Statistical summary of numeric cols
03 | Pandas

Selecting Features

# Single column selection
ages = df["Age"]

# Multiple column selection
subset = df[["Name", "Salary"]]
03 | Pandas

Filtering Observations

# Boolean indexing
high_earners = df[df["Salary"] > 55000]

# Combining conditions
seniors = df[(df["Age"] > 30) & (df["Salary"] > 60000)]
03 | Pandas

Handling Null Data

# Find missing values
df.isnull().sum()

# Fill missing ages with the average
df["Age"] = df["Age"].fillna(df["Age"].mean())
03 | Pandas

Feature Creation

# Simple arithmetic on columns
df["Tax_Amount"] = df["Salary"] * 0.2
df["Net_Salary"] = df["Salary"] - df["Tax_Amount"]
03 | Pandas

Grouping & Aggregation

# Calculate mean salary per department
avg_salary = df.groupby("Department")["Salary"].mean()
04 | Cleaning

Removing Duplicates

# In-place modification removes redundancy
df.drop_duplicates(inplace=True)
04 | Cleaning

Encoding Categories

ML Models can't read "Sales" or "HR". We must convert them to numbers.

# One-Hot Encoding
df = pd.get_dummies(df, columns=["Department"])
04 | Cleaning

Feature Scaling

# Standardizing using NumPy functions
mean = np.mean(df["Age"])
std = np.std(df["Age"])

df["Age_Scaled"] = (df["Age"] - mean) / std
04 | Cleaning

Outlier Detection (IQR)

Q1 = df["Salary"].quantile(0.25)
Q3 = df["Salary"].quantile(0.75)
IQR = Q3 - Q1

# Define bounds
lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

outliers = df[(df["Salary"] < lower) | (df["Salary"] > upper)]

08 | Real Example

End-to-End Analysis

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# 1. Load
df = pd.read_csv("sales_data.csv")

# 2. Clean
df.dropna(inplace=True)

# 3. Explore
print(df.describe())

# 4. Visualize
sns.histplot(df["Revenue"], kde=True)
plt.title("Distribution of Revenue")
plt.show()
09 | Best Practices

Performance Tips

  • Vectorize: Never use for loops to iterate over DataFrames.
  • Types: Use df.info() to ensure dates are actually datetime types.
  • Memory: Downcast floats if working with massive datasets.
09 | Best Practices

Visual Integrity

Technical: Seaborn Heatmaps.
Executive: Plotly Dashboards.
