Reducing Memory Usage

Advanced

The IQR Method

Q1 = df['revenue'].quantile(0.25)
Q3 = df['revenue'].quantile(0.75)
IQR = Q3 - Q1

lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

# Find outliers
outliers = df[(df['revenue'] < lower) | (df['revenue'] > upper)]
Advanced

Handling Outliers

  • Remove: Only if they are data errors[cite: 15].
  • Winsorize: Cap values at percentile bounds[cite: 15].
  • Log Transform: Compress skewness[cite: 15].
