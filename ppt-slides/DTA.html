Chapter 02

Exploratory Data Analysis (EDA)

EDA is the process of using visualization and statistics to understand the patterns and anomalies in your data before modeling.

EDA

Single Variable Analysis

Checking the distribution of a single feature:

import matplotlib.pyplot as plt
import seaborn as sns

sns.histplot(df['age'], kde=True)
plt.show()
EDA

Categorical Distribution

print(df['department'].value_counts())
sns.countplot(x='department', data=df)
EDA

Correlation Analysis

How do two variables relate? Use a Scatter Plot.

sns.scatterplot(x='age', y='salary', data=df)
EDA

The Heatmap

Visualizing the correlation matrix of all features:

correlation_matrix = df.corr()
sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm')
EDA

Identifying Outliers

Boxplots are essential for spotting data points that fall far outside the norm.

sns.boxplot(x=df['salary'])
EDA

The Null Count

# Check total missing per column
print(df.isnull().sum())
Preparation

Feature Creation

Creating new information from existing columns:

df['salary_per_year'] = df['monthly_salary'] * 12
EDA

Working with Dates

df['date'] = pd.to_datetime(df['timestamp'])
df['year'] = df['date'].dt.year
EDA Checklist

Is your Data Understood?

Chapter 03

Data Cleaning & Handling

Garbage In, Garbage Out. Cleaning is 80% of an ML Engineer's work.

Cleaning

Dropping NaNs

# Remove any row with a missing value
df_clean = df.dropna()

# Only drop if specific column is missing
df_clean = df.dropna(subset=['email'])
Cleaning

Filling The Gaps

Instead of deleting, we can fill missing data with the Mean or Median:

df['age'] = df['age'].fillna(df['age'].mean())
Cleaning

Removing Duplicates

print(df.duplicated().sum())
df = df.drop_duplicates()
Cleaning

Text Standardization

df['name'] = df['name'].str.strip().str.title()
Cleaning

Fixing Data Types

# Convert object to float
df['price'] = pd.to_numeric(df['price'], errors='coerce')
Manipulation

Sorting Data

df_sorted = df.sort_values(by='salary', ascending=False)
Aggregation

The GroupBy Operation

Summarize data by categories:

dept_avg = df.groupby('department')['salary'].mean()
Aggregation

Multiple Aggregates

df.groupby('dept').agg({'salary': 'mean', 'age': 'max'})
Output

Exporting Results

df.to_csv('cleaned_data.csv', index=False)
Chapter 04

Advanced Acquisition: Web Scraping

Sometimes data isn't in a file; it's on a webpage. We use BeautifulSoup to extract it.

Scraping

Fetching HTML

import requests
from bs4 import BeautifulSoup

url = 'https://example.com/data'
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')
Scraping

Finding Tags

# Get all headings
titles = soup.find_all('h2')

# Get element by class
prices = soup.find_all('span', class_='price-tag')
Scraping

Scraping Tables

Pandas has a shortcut for HTML tables:

tables = pd.read_html(url)
df_from_web = tables[0]
Important

The Ethics of Scraping

Always check site.com/robots.txt before scraping. Do not overwhelm servers with too many requests per second.
Architecture

Building Pipelines

A pipeline is a series of functions that transform raw data into a model-ready format.

Architecture

The Acquisition Class

class DataPipeline:
    def load(self, path):
        self.df = pd.read_csv(path)
        return self
    
    def clean(self):
        self.df = self.df.drop_duplicates().dropna()
        return self
    
    def get(self):
        return self.df
Architecture

Fluent Interface

pipeline = DataPipeline()
df = pipeline.load('raw.csv').clean().get()
Professionalism

Log Your Progress

Always print the shape of your dataframe before and after cleaning. If you go from 10,000 rows to 10 rows, your cleaning logic is too aggressive!
